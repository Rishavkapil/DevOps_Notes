Transport layer Security (TLS) encrypts data sent over internet to ensure that hackers are unable to see what you transmit is particularly useful for private and sensitive information such as passwords, credit card numbers , and personal correspondence.  

### What is TLS?
TLS is a cryptographic protocol that provides end-to-end security to the data sent over the internet. 

It should be noted that TLS does not secure data on end systems. It simply ensures the secure delivery of data over the internet. 

### How does TLS work ?

TLS uses combination of symmetric and asymmetric cryptography, as this provides good compromise between performance and security when transmitting data securely.

With Symmetric Cryptography, data encrypted and decrypted with the secret key known to both sender and recipient. 

Asymmetric cryptography uses key pairs - a public key and private key . The public key is mathematically related to private key.
