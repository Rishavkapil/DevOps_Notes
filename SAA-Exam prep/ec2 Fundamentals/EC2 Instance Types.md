
## On-demand instances

Short workload , predictable pricing, pay per use. 

## Reserved Instances

- Can be purchased only for 1 or 3 years , no in-between

- Used for Long workloads. 

- Upto 72% discount as compared to on-demand instances. 
**Convertible Reserved Instances**: Long workloads with flexible instances. 


## Savings Plan

- for 1 and 3 years. 
- Commitment for an amount of usage
- For long workloads. 

## Spot Instances

- For short workloads
- cheap
- can get discount upto 90% as compared to on-demand
- Can loose instances
- Use when batch jobs, data analytics etc. 
## Dedicated Hosts

- Book an entire physical server


## Spot Fleet 

Set of spot instances + on-demand instances
