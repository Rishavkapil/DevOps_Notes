DDOS : Distributed Denial of Service  - many requests at the same time. 

### AWS Shield Standard

- Free service that is activated for every customer.
- Provides protection from attacks such as SYN/UDP floods, reflection attacks or other layer 3/4 attacks. 

### AWS Shield Advanced 

- Optional DDoS mitigation service($3000 per month per organization)
- Offers more advanced features and more security, 24/7 security and also protects against attack at layer 7.
